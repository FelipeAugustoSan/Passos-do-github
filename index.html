  1. Passos para hospedar um Projeto no GITHub Pages
  2. No terminal, inicialize om repositorio local com git init
  3. Altere o nome da branc preincipal para main
  4. Adicione os arquivos com Git add . e faça o commit git vommit -m ""first commit
  5. Conecte ao repositorio remoto com:
    git remote add origin https//github.com/SEU_Usuario/NOME+DO_REPOSITORIO.gti
  6. Envie os arquivos com git push -u origin main
  7. No github, vá em Settings > Pages
  8. Em "Branch", selecione maine a posta /(root)Clique em Save
  9. Após alguns segundos, seu site estará disponível no link exibido em "Pages"
